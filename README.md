# Mesaj de scuze

Acesta este un proiect GitHub simplu pentru a transmite un mesaj sincer de scuze și motivele pentru care merită să luptați pentru relația voastră.

## Conținut

- `index.html` — pagina principală cu mesajul de scuze, motive și promisiuni.
- `styles.css` — stiluri responsive și o prezentare elegantă.

## Cum folosești

1. Clonează sau descarcă acest repository.
2. Deschide `index.html` în browser.
3. Modifică textul din pagină cu numele și detaliile personale.

## Publicare

Dacă vrei să afișezi pagina online, poți folosi GitHub Pages din setările repository-ului.
